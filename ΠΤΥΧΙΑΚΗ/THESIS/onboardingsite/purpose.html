Frequently Asked Questions

This extension is designed to enhance web accessibility by providing tools and features that make browsing the internet easier for individuals with disabilities.

To install the extension, visit the Chrome Web Store and search for our extension by name. Click the "Add to Chrome" button on the extension's page and follow the on-screen instructions to complete the installation process.

AThe extension offers various features such as customizable font sizes, removal of images and videos, toggling saturation levels, and applying a dyslexia-friendly font, all aimed at enhancing web accessibility for users.

Currently, the extension provides preset features to enhance accessibility. We are working on adding more customizable settings in future updates based on user feedback.
